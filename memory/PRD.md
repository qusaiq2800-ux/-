# Random Picks - موقع الاقتراحات العشوائية

## المشكلة الأصلية
موقع يساعد المستخدم عندما يكون محتارًا في ماذا يلعب أو يشاهد، عن طريق اقتراح عشوائي من أربع فئات رئيسية:
- ألعاب
- أفلام  
- مسلسلات
- فيديوهات يوتيوب

**المتطلبات الرئيسية:**
- أسماء حقيقية وليست خيالية
- عدد كبير من الخيارات لكل فئة
- عدم تكرار الاقتراحات

## البنية التقنية
- **Frontend:** React 19 + TailwindCSS + Shadcn UI
- **Backend:** FastAPI + Motor (Async MongoDB)
- **Database:** MongoDB
- **Design:** Dark theme, RTL Arabic, Glassmorphism effects

## APIs المتاحة
- `GET /api/categories` - جميع الفئات مع الأعداد
- `GET /api/suggest/{category}?genre=X&exclude_ids=X` - اقتراح عشوائي مع فلترة
- `GET /api/genres/{category}` - جميع الأنواع لفئة معينة
- `GET /api/favorites` - قائمة المفضلة
- `POST /api/favorites` - إضافة للمفضلة
- `DELETE /api/favorites/{item_id}` - حذف من المفضلة
